Multi-camera Human Tracking

Tracking people within a video is not a new problem. It has been tackled by a lot of people in the past and several solutions exist. While tracking people among multiple surveillance cameras is a relatively new problem, it is hard to get right and is even a cumbersome process to do manually. Specifically, re-identifying people across multiple cameras is difficult as it often results in change in posture, shape and lighting. Many of these problems have resurfaced due to advancements in deep learning which has made these seemingly unsolvable problems solvable. In this report we present our approach to solving this problem using the latest technologies. Specifically, we perform person detection using a pre-trained deep learning model YOLOv4, then we perform tracking across a single camera using Kalman Filter. Lastly, we train a re-identification model on the Market-1501 dataset using EfficientNet as a backend. The model achieves rank-1 accuracy of 94.27%. Then we present the results and findings by using this model in inference. We also present an application which can be used to retrieve people from the videos.
